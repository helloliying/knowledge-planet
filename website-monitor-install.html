🌐 网站行为监控系统

安装指南：监控 https://helloliying.github.io/knowledge-planet/ 及其所有下级页面的用户行为

⚠️ 重要提示

本监控系统用于收集用户行为数据，请确保遵守相关隐私法律法规，并在网站隐私政策中说明数据收集情况。

1
下载监控脚本

首先，下载网站监控脚本文件：

// 监控脚本文件
文件位置: /static/js/website-monitor.js
大小: ~13KB
功能: 收集鼠标移动、点击、滚动、页面访问等行为数据
📥 下载监控脚本
2
添加到网站页面

将以下代码添加到网站所有页面的 <head> 标签中：

<!-- 网站行为监控脚本 -->
<script src="/knowledge-planet/js/website-monitor.js"></script>

对于Hugo网站，可以添加到主题模板中：

{{/* 在 layouts/_default/baseof.html 或类似模板文件中 */}}
<head>
  <!-- 其他head内容 -->
  <script src="{{ "js/website-monitor.js" | absURL }}"></script>
</head>
3
访问监控仪表板

安装完成后，访问监控仪表板查看收集的数据：

// 监控仪表板地址
https://helloliying.github.io/knowledge-planet/website-monitor-dashboard.html
📊 打开监控仪表板 🔒 打开增强版仪表板

📈 全面监控

监控鼠标移动、点击、滚动、页面访问、表单交互、链接点击等全方位用户行为

🌍 跨页面跟踪

跟踪用户在整个网站的所有页面间的行为，形成完整的用户旅程

🔒 本地存储

数据存储在用户的浏览器本地存储中，保护用户隐私

📊 丰富可视化

提供事件分布、页面访问统计、鼠标轨迹热力图等多种可视化图表

💾 数据导出

支持JSON格式数据导出和下载，便于进一步分析

⚡ 轻量高效

监控脚本仅13KB，对网站性能影响极小

🔧 配置选项

监控脚本支持以下配置选项（在 website-monitor.js 中修改）：

const CONFIG = {
  // 基础配置
  enabled: true, // 是否启用监控
  sampleInterval: 50, // 采样间隔(ms)
  maxRecords: 10000, // 最大记录数

  // 存储配置
  storageKey: 'ks_website_monitor',
  autoSave: true,
  saveInterval: 5000, // 自动保存间隔(ms)

  // 调试
  debug: false, // 启用调试日志
  logPrefix: '[网站监控]'
};
