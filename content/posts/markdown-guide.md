# Markdown写作指南

Markdown是一种轻量级标记语言，让写作变得简单高效。以下是常用语法：

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本样式

- **粗体**：`**粗体**` 或 `__粗体__`
- *斜体*：`*斜体*` 或 `_斜体_`
- ~~删除线~~：`~~删除线~~`
- `行内代码`：\`行内代码\`

## 列表

### 无序列表
```markdown
- 项目一
- 项目二
  - 子项目
```

### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```

## 链接和图片

- 链接：`[链接文字](https://example.com)`
- 图片：`![图片描述](图片地址)`

## 代码块

\```语言
代码内容
\```
